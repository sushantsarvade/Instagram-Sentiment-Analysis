### Instagram Sentiment Analysis

## Project Overview

The project analyzes Instagram comments to identify user sentiment. Using Python and data analysis techniques, comments are cleaned, processed, and categorized into positive, negative, and neutral sentiments. The analysis helps understand user reactions and engagement patterns.

## Objectives

- Clean and preprocess Instagram comment datasets

- Perform exploratory data analysis (EDA) on user comments

- Identify sentiment patterns in comments

- Categorize comments based on sentiment polarity

- Ensure dataset quality and consistency during analysis

## Tools Used in the project

- Python

- Pandas

- Matplotlib

- TextBlob (for sentiment analysis)

## Dataset

The dataset contains Instagram comments collected from public posts. It includes user comments that are processed and analyzed to determine sentiment patterns.

Example column:

Text
This post is amazing
I love this content
Worst update ever


## Project Workflow

- Data Collection – Gather Instagram comment data

- Data Cleaning – Remove unnecessary characters and normalize text

- Exploratory Data Analysis – Examine comment patterns and sentiment distribution

- Sentiment Classification – Categorize comments into positive, negative, and neutral

- Visualization – Generate charts to display sentiment distribution

## Results

The analysis provides insights into user sentiment by classifying comments and visualizing the distribution of positive, negative, and neutral responses.
